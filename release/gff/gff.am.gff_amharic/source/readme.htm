Amharic Lexical Model

This is an Amharic (አማርኛ , ISO-639-2 am) lexical model developed to support predictive text in the corresponding GFF Amharic keyboard. The lexical model relies on the UniLex projects's word List for Amharic. The word list has been modified to remove English entries only but has not otherwise been reviewed to validate spelling. Spelling refinements and the incorporation of additional word lists are goals of future development of this lexical model.
